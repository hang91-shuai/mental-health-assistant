# Data 目录说明

本目录包含心理健康助手项目的数据资源，分为两大部分：
- **datasets/** — 中文情绪分类标注数据集
- **psychology_kb/** — 心理学知识库文档

## 四、文件命名规范

知识库文件统一使用以下命名格式：

```
{来源前缀}_{主题描述}_{日期}.txt
```

**来源前缀：**
- `who_` — 世界卫生组织
- `yixinli_` — 壹心理
- `dxy_` — 丁香医生/丁香园
- `jiandanxinli_` — 简单心理
- `zhihu_` — 知乎

**示例：**
- `who_mental_health_concept_20260722.txt`
- `dxy_depression_disorder_20260722.txt`
- `yixinli_procrastination_adhd_20260722.txt`
